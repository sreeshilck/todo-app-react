
# Todo App - React

A Todo Application in React.js




## Features

- Add , edit, delete todo
- Responsive


## Tech Stack

- React.js
- uuid


## Run 

Clone the project

```bash
  git clone https://github.com/sreeshilck/todo-app-react
```

Go to the project directory

```bash
  cd todo-app-react/
```

Install dependencies

```bash
  npm install
```

Start 

```bash
  npm run start
```
